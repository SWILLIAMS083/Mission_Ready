Mission 1 Code Breakdown

Home About Code

1. Linking External CSS

This keeps structure separate from styling.


<link rel="stylesheet" href="styles.css">

2. Navigation Between Pages

Using anchor tags to connect HTML files.


<a href="index.html">Home</a>
<a href="about.html">About</a>

3. Reusable Layout with Classes

Classes allow consistent styling across sections.


<div class="card">
  <h2>Section Title</h2>
  <p>Reusable content block</p>
</div>

4. Contact Form

Using form, label, input, textarea and button elements.


<form action="mailto:you@example.com">
  <label>Name</label>
  <input type="text">
  <textarea></textarea>
  <button>Send</button>
</form>

5. Flexbox Layout

Flexbox controls alignment and spacing.


.nav {
  display: flex;
  gap: 12px;
}

6. CSS Animation

Keyframes define animation, class applies it.


@keyframes pulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.03); }
}

.btn {
  animation: pulse 2s infinite;
}
