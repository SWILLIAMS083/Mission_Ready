Why I Built It This Way

  • This is a multi-page site using separate HTML files.
  • Both pages link to the same external stylesheet.
  • This keeps structure (HTML) separate from styling (CSS).
  • The layout is reusable using shared classes.
  • The navigation links connect the pages together.
